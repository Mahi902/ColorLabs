Welcome to the "asset" directory. Every image, graph, and icon used within Color Labs are stored here as assets. The asset names follow a numeric structure, with the file extension at the end. These numeric values and the file extensions are called "asset-id"s. If you want to fetch the files, use urls as: "https://mahi902.github.io/ColorLabs/asset/{asset-id}". A list of all the assets under this directory can be found at "https://mahi902.github.io/ColorLabs/asset.html".
